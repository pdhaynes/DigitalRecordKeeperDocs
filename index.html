Digital Record Keeper

About

The Digital Record Keeper App is a secure and easily accessible repository for sensitive user files and documents.

Encrypted

All files are secured with military grade AES encryption and have unique keys generated on a per file basis.

The Only Downside is Complete Ownership

The user has the ultimate control and ownership over all files. The app simply serves as an encryption medium for sensitive user data, This means:

  • All information is stored and encrypted locally on YOUR device.
  • Files/encryption keys are NOT stored in remote locations, such as the cloud.
  • File access passwords are hashed and stored locally in an encrypted Database

This sounds great, where is the a downside?

Since the user is given complete control, we do not have access to any of the data you store. This means we are unable to:

  • Reset File Passwords
  • Reset User Passwords
  • Decrypt Files
  • Retrieve Encryption Keys

While this may sound scary at first, we are committed to enabling the user to "help themselves" as many ways as feasibly possible.
